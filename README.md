# Installation
### Clone this repository
```
git clone https://github.com/Durian-Inc/Huh
```

### Install dependencies
* Python 3.X
* pip3 (python3-pip)
* node js

### Additional installation instructions
#### API
1. `cd api`
2. `python3 run.py`

#### Front End
1. `cd react-web`
2. `yarn`
3. `yarn dev`

#### Chrome Extension
1. Open Chrome
2. Go to "Extensions"
3. Turn on "Developer mode"
4. Click "Load unpacked"
5. Navigate to `chrome-extension`
6. Confirm
