# Installation
### Clone this repository
```
git clone https://github.com/Durian-Inc/Huh
```

### Install dependencies
* Python 3.X
* pip3 (python3-pip)
* node js

### Additional installation instructions
#### API
* `cd api`
* `python3 run.py`

#### Front End
* `cd react-web`
* `yarn`
* `yarn dev`

#### Chrome Extension
* Open Chrome
* Go to "Extensions"
* Turn on "Developer mode"
* Click "Load unpacked"
* Navigate to `chrome-extension`
* Confirm
